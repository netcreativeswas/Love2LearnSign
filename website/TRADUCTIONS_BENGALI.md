## Page d'accueil

**Titre principal:** বাংলা ইশারা ভাষা শিখুন (Learn Bangla Sign Language)

**Tagline:** শিখুন। অনুশীলন করুন। মনে রাখুন। (Learn. Practice. Remember.)

**Description:** সংক্ষিপ্ত ভিডিও এবং একটি সহজ শেখার প্রবাহের মাধ্যমে বাংলা ইশারা ভাষা আবিষ্কার করুন। অভিধান, ফ্ল্যাশকার্ড, কুইজ, পছন্দসই এবং স্পেসড রিপিটিশন দিয়ে ধাপে ধাপে শব্দভান্ডার গড়ে তুলতে অনুশীলন করুন। ইংরেজি এবং বাংলায় উপলব্ধ।

**Bouton CTA:** সহায়তার সাথে যোগাযোগ করুন (Contact support)

**Fonctionnalités:**
- **অভিধান** (Dictionary): দ্রুত সাইন খুঁজুন এবং পছন্দসই সংরক্ষণ করুন।
- **কুইজ** (Quizzes): সময়-সীমাবদ্ধ, বিভাগ-ভিত্তিক কুইজ দিয়ে অনুশীলন করুন।
- **ফ্ল্যাশকার্ড** (Flashcards): শব্দ মনে রাখতে স্পেসড রিপিটিশন।
- **অফলাইন** (Offline): স্মুথার প্লেব্যাকের জন্য ক্যাশ করা ভিডিও।

## Comment modifier les traductions

Toutes les traductions sont dans `website/src/locales/bn.json`. Vous pouvez modifier ce fichier directement pour corriger ou améliorer les traductions.

Le système détecte automatiquement les changements et les applique au site.

## URLs

- Anglais (par défaut): `/`, `/contact`, `/donate`, etc.
- Bengali: `/bn/`, `/bn/contact`, `/bn/donate`, etc.

Le sélecteur de langue dans le header permet de basculer entre les deux langues.
